# Web Games

A collection of web-based games built using HTML, CSS, and JavaScript.

## Games Included
- Nine Men's Morris
- Memory Game
- Rock Paper Scissors Lizard Spock

## Features
- Interactive browser gameplay
- Responsive styling
- Score tracking and game logic in JavaScript

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Open the home html file (**`index.html`**) or desired html file in a browser
